Python

I have learned Python systematically and applied it in several class projects as well as my internship. Throughout my studies, I had an idea to build a software for my dad. He is a clothing manufacturer, so I decided to design an inventory management software.

After consulting with my father, based on the real scenario, I created several personas, and according to their problems and requirements, I identified the issues, and set those as the requirements that main features needed to address.

This project's prime emphasis and motivation are continuously improving the software features by investigating the desires and exploring the user's experience. The purpose of the program is to be user-friendly, and most importantly, improve the efficiency and competitiveness of enterprises through inventory improvement. Exploring and discovering different programming languages and deciding on a development environment for the software was my initial challenge. Moreover, with the concept of simplicity and usability, I need to design a user-friendly and straightforward interface. Lastly, creating a comprehensive database capable of storing a wide variety of data, such as high-quality images.

Because of the short developing time period, I have to focuse more on the functional design instead of the user interface. And in order to finish it faster the whole project was done with my primary programming language. In the process of development, I also found out some limitations with Python in front-end. Therefore, I am going to build a web and mobile version based on this project.
